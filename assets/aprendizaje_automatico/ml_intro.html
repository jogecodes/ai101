Machine Learning: Introducción

El objetivo de las técnicas de Machine Learning (ML) es proporcionar un marco a través del cual se pueda dar el salto de un conjunto de variables predictoras con una etiqueta (o una serie de ellas) a un modelo de clasificación, para así conseguir hacer predicciones sobre elementos que sólo contengan información de las variables predictoras.

Los modelos clasificadores generados a partir de una serie de NNN﻿ instancias de datos caracterizadas por un conjunto de variables predictoras (características) y otro conjunto de variables de clase (etiquetas) se engloban en las técnicas de aprendizaje supervisado.

La pareja (x(i),c(i))(\textbf x^{(i)},c^{(i)})(x(i),c(i))﻿ constituye un dato (o instancia) del problema.

Frente a las técnicas de aprendizaje supervisado existen las de aprendizaje no supervisado, en las que el modelo se genera a partir de NN N﻿ instancias de datos que están sólamente caracterizadas por las variables predictoras (no existen variables de clase).

El objetivo de los modelos clasificadores de aprendizaje no supervisado es agrupar las instancias en clusters de modo que haya alta variabilidad entre diferentes clusters (las instancias pertenecientes a diferentes clusters sean muy distintas) y muy baja variabilidad entre instancias clasificadas en un mismo cluster.

También existen problemas de clasificación semisupervisada, cuyo objetivo es resolver de forma eficiente la falta de etiquetas en algunos de los datos del set, buscando mejorar el resultado que se obtendría con respecto al uso de sólo los datos completos (etiquetados) en un algoritmo de aprendizaje supervisado.

A grandes rasgos se pueden distinguir dos escuelas de clasificación automatizada:

Modelado estadístico

Asume que los datos vienen generados de forma estocástica, y procede seleccionando un modelo y ajustándolo a los datos, lo que proporciona un resultado probabilístico.

Machine learning

Emplea modelos algorítmicos, tratando como desconocido el mecanismo que genera los datos. Procede aprendiendo una estructura y una serie de parámetros para el clasificador, y proporciona un resultado determinista.
