Clasificación supervisada unidimensional

El problema de clasificación supervisada unidimensional consiste en, a partir de un conjunto de datos correspondientes a instanciaciones de un conjunto de variables predictoras, cada instanciación asociada a una variable de clase (etiqueta), elaborar un modelo que permita asignar un valor a la variable de clase dada una nueva serie de datos comprensivos de la instanciación de las demás variables:

Para ello el modelo de clasificación emplea una función clasificadora ϕ(x)\phi(\mathbf x)ϕ(x)﻿ que devuelve una etiqueta para un valor concreto de las variables a clasificar.

La función clasificadora debe dividir el espacio de instanciaciones en regiones de decisión, una por cada etiqueta de clase, de modo que si un dato x\textbf xx﻿ cae en la región asociada con ccc﻿, su etiqueta será ϕ(x)=c\phi (\textbf x)=cϕ(x)=c﻿. Tales regiones están separadas por fronteras de decisión, hipersuperficies en el espacio de instanciaciones, correspondiente a aquellos argumentos de ϕ\phiϕ﻿ que devuelven dos etiquetas distintas.

Cuanto más flexibles sean las fronteras de decisión, mejor rendimiento tendrá el clasificador.

Los modelos de multired bayesiana (Geiger y Heckerman, 1996) permiten además codificar independencias condicionales asimétricas, aquellas que sólo se verifican para ciertas instanciaciones de las variables involucradas, pero no para todas ellas.

Las redes bayesianas locales que componen la multired están asociadas, por tanto, a un subconjunto de una partición del dominio de la variable HHH﻿.
